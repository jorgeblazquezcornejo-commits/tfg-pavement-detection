## Dataset

Debido al gran tamaño de los conjuntos de datos utilizados, estos no se incluyen en el repositorio.

El entrenamiento se realiza sobre el dataset RDD2022 (Road Damage Dataset 2022), disponible públicamente en:
https://github.com/sekilab/RoadDamageDetector

Las imágenes deben organizarse siguiendo la estructura indicada en los archivos YAML de configuración.

## Ejecución

El notebook `gcolab_train_YOLOv8.ipynb` está diseñado para ejecutarse en Google Colab y asume que el dataset se encuentra montado en Google Drive.
